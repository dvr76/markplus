A browser fingerprinting library for device identification and fraud detection.

Fork Notice: This project is a fork of ThumbmarkJS by Ilkka Peltola. Full credit to the original ThumbmarkJS project for the core fingerprinting technology. This is a simple side project that extends the MIT-licensed version with additional client-side detection capabilities.

What is markplus?

markplus is a client-side JavaScript library that generates unique browser fingerprints based on browser characteristics and capabilities. It can be used for:

  • Device identification
  • Fraud detection
  • Bot detection
  • User tracking (with proper consent)
  • Security analysis

Features

Built on the original ThumbmarkJS foundation, markplus includes:

  • Canvas and WebGL fingerprinting
  • Audio context fingerprinting
  • Font detection
  • Screen characteristics
  • Browser permissions analysis
  • TLS/cipher suite capabilities
  • HTTP protocol support detection
  • HTTP headers and client hints

Key Differences from Original

This fork adds the following client-side features:

  • TLS Detection: Fingerprints cryptographic capabilities
  • Protocol Detection: Detects HTTP/2, HTTP/3, WebSocket, WebRTC support
  • Enhanced Headers: Client Hints, Accept headers, security headers
Note: All fingerprinting is performed client-side. This library does not connect to any external services or APIs.

Quick Start

See the interactive example to test the library in your browser.

To use in your project:

  1. Build the library: npm run build
  2. Include the appropriate dist file in your HTML
  3. Call the fingerprinting function
